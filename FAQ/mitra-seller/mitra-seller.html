Cara atur nama dan alamat toko

  • Buka link bargros.id
  • Silahkan login menggunakan akun Mitra Seller yg sudah anda daftar kan di Bargros.id
  • Klik Menu Dashboard
  • Ketika masuk ke dashboard
  • klik tombol Pengaturan
  • Klik tombol Toko
  • Setelah itu anda bisa lengkapi formulir kebutuhan toko anda sebagai mitra seller
  • ketika sudah lengkap, Klik tombol Update

Cara upload produk sebagai mitra seller

  • Kita lanjut untuk Upload Produk sebagai Mitra Seller
  • Klik menu Produk > Tambah Produk
  • Setelah itu anda akan masuk ke halaman Upload Produk seperti ini
  • Lengkapi semua formulir untuk produk anda
  • Setelah terisi semua formulir untuk produk anda
  • Anda bisa klik tambah variant *jika perlu / jika produk anda mempunyai variant seperti Bahan, Warna, Ukuran dll
  • Akan muncul beberapa default variant seperti Warna, Ukuran, Bahan Anda juga bisa custom variant disini, tinggal klik lalu enter
  • Jika tidak ada variant untuk produk anda, anda bisa langsung kli tombol Tambah Produk
  • Jika anda berhasil upload produk akan muncul produk yg baru saja anda tambahkan
  • Perlu di perhatikan ketika anda sudah berhasil upload produk tapi ga muncul produk nya di halaman utama Bargros.id anda bisa liat status yg sudah di kasih tanda panah itu yg artinya requested sedang di cek
  • Anda juga bisa menambahkan Produk Baru tinggal klik tombol Tambah Produk Lalu seperti biasa lengkapi formulir produk dan selesai

Cara menambahkan warehouse sebagai mitra seller

  • Klik menu Gudang
  • Setelah itu anda akan di arahkan ke halaman Gudang
  • Klik tombol Tambah Warehouse untuk menambahkan warehouse baru
  • Anda akan di arahkan untuk mengisi formulir warehouse baru
  • Lengkapi formulir warehouse yg baru
  • Anda juga bisa ubah status Warehouse untuk di Aktifkan/Tidak
  • Kalo sudah lengkap formulir yg di butuhkan untuk menambahkan warehouse, kamu bisa langsung klik tombol Tambah Warehouse
  • Setelah anda klik tombol tambah warehouse anda akan di bawa ke halaman detail warehouse yg baru saja anda daftarkan
  • Jika ada yg typo/salah ketik tenang, anda bisa klik tombol Edit untuk mengubah data warehouse yg baru

Cara memproses pesanan yang masuk

  • Buka link bargros.id
  • Silahkan login menggunakan akun Mitra Seller yg sudah anda daftar kan di Bargros.id
  • Klik Menu Dashboard
  • Ketika sudah masuk dashboard
  • Lihat fitur dengan judul perlu di verifikasi
  • Disitu juga akan terlihan ada berapa pesanan yg perlu di verifikasi
  • Klik tombol More Info
  • Kalo sudah, ada beberapa informasi terkait pesanan dari barang yg sudah kamu jual di Bargros.id
  • Klik tombol Verifikasi untuk memverifikasi pesanan pelanggan yg sudah bayar
  • Atau anda bisa klik tombol Detail untuk melihat lebih detail mengenai pesanan yg di buat oleh pelanggan
  • Ketika sudah berhasil proses verifikasi pesanan
  • Anda bisa klik tombol menu Perlu Dikirim
  • Lalu anda bisa klik tombol Atur Pengiriman akan muncul popup untuk memasukan Nomor Resi lalu klik tombol Set Pengiriman, selesai
  • Ada beberapa tombol yg mungkin bisa membantu anda mulai dari
  • Cetak Resi
  • Detail
  • Atur Pengiriman
  • Jika pesanan sudah di terima oleh Pelanggan Anda bisa cek tab Dikirim Untuk memberi tanda bahwa barang sudah Selesai

Cara menambahkan user baru

  • Masuk ke Dashboard Bargros.id
  • Klik menu Users
  • Ketika sudah, anda akan di arahkan ke halaman tambah user
  • Klik tombol Tambah User
  • Silahkan anda isi Formulir kebutuhan menambah user baru
  • Perhatikan tanda panah untuk Setting user
  • Setelah formulir lengkap anda bisa klik tombol Tambah User
